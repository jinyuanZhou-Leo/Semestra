Semestra is a full-stack scheduling and academic planner with a Python backend and modern web frontend.
`backend/` provides FastAPI APIs, data models, persistence, schema-level widget layout validation, and env-backed JWT signing plus HttpOnly auth-cookie session transport.
`frontend/` provides the user interface, including RGL v2 container-width-based widget sizing, lower-frequency resize-guarded relayouts, zero-width mount protection on dashboard remount, optional aspect-ratio constraints, context-safe optimistic dashboard sync, stale-response-safe fetch flows, semantic-token dashboard FAB surfaces tuned with light-mode light-green glass active edit state, deep-green dark mode, and shadowless dark mode, class-driven dark-variant behavior aligned with `ThemeProvider`, softened lower-opacity widget header controls with direct touch edit-mode actions, single-ring card-aligned widget chrome with edit-mode-only hover elevation, skeleton-first plugin loading with fade-in completion for tabs/widgets, simple tab loading skeletons, refined habit-streak Duolingo/classic-ring widgets with shared streak data, split runtime definitions, cell-targeted Duolingo feedback, lighter burst effects, and single-shell chrome, plus hardened plugin UX around explicit course-list feedback, a FullCalendar-backed builtin-event-core calendar that now consumes a standalone `calendar-core` source registry, scroll-first small-screen behavior, lighter unwrapped today emphasis, a circular month-day badge, a more compact Calendar skeleton, draft-friendly time-window inputs, timezone-safe semester date parsing, semester Reading Week configuration with optional Monday-to-Sunday selection, Reading Week event suppression, shifted week-number labeling, registry-backed schedule/todo overlays with all-day support and live refresh, lighter Todo tab loading shells, compact workspace-level semester/course navigation rows that keep breadcrumb context in the global header while moving titles and tab switching into a second nav layer, settings-card sticky titles that now track live workspace-header height, dashboard-only overview stat strips with normal labels, optional icons, and tighter mobile height, stable right-aligned semester/course homepage tab selectors, extra-large mobile semester/course titles, matched calendar shell/scrollgrid radii, dark-mode-aware scrollbars, shadcn-aligned event cards/conflict styling with slim accent rails, subtle corner radii, compact badges, explicit clickable hover/active feedback, full-width single-event lanes, no top conflict summary badge, remount-based view syncing, conflict details, guarded todo flows, validated counter ranges, locale-aware world-clock formatting, and cookie-backed auth bootstrapping that removes browser token persistence in localStorage.
