`backend/` provides FastAPI APIs, data models, persistence, schema-level widget layout validation, and env-backed JWT signing plus HttpOnly auth-cookie session transport.
`frontend/` provides the user interface, including RGL v2 container-width-based widget sizing, lower-frequency resize-guarded relayouts, zero-width mount protection on dashboard remount,
